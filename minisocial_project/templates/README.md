# MiniSocial — Pet-проект на Django

Интерактивная мини-социальная сеть. 

### Функционал:
* **Пользователи**: Регистрация, авторизация, профили.
* **Посты**: Создание, чтение, редактирование и удаление (CRUD).
* **Комментарии**: Возможность комментировать публикации.
* **Безопасность**: Доступ к редактированию только для авторов.

### Технологии:
* Python 3.x, Django 5.x
* Bootstrap 5 (Frontend)
* SQLite (Database)
