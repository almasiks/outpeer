# Образовательная платформа "KBTU Tutor"

## Описание проекта
**KBTU Tutor** — это веб-приложение для студентов КБТУ, которое помогает найти тьюторов среди сокурсников. Платформа позволяет бронировать время для консультаций по сложным предметам, помогать с лабораторными работами и готовиться к экзаменам.
